3rd Party Assemblies Used
================================

OpenCV
	Desc:		Emgu CV is a cross platform .Net wrapper to the OpenCV image processing library.
	Homepage:	www.emgu.com/wiki/index.php/Main_Page
	Download:	https://sourceforge.net/projects/emgucv/files/latest/download?source=files

Network Tables
	Desc:		FRC NetworkTables for .NET 
	Homepage:	https://github.com/robotdotnet/NetworkTables
	Download	nuget

SimpleTCP
	Desc:		.NET library to handle the repetitive tasks of spinning up and working with TCP sockets (client and server).
	Homepage:	https://github.com/BrandonPotter/SimpleTCP
	Download:	nuget

System.ValueTuple
	Desc:		Provides the System.ValueTuple structs, which implement the underlying types for tuples in C# and Visual Basic.
	Homepage:	https://www.nuget.org/packages/System.ValueTuple/
	Download:	nuget

TopShelf
	Desc:		Topshelf is a framework for hosting windows services written using the .NET framework.
	Homepage:	http://topshelf-project.com/
	Download:	nuget

BlinkStick
	Desc:		Smart LED controllers with integrated USB firmware.
	Homepage:	https://www.blinkstick.com/
	Download:	https://github.com/arvydas/BlinkStickDotNet

JSON.NET
	Desc:		Popular high-performance JSON framework for .NET
	Homepage:	https://www.newtonsoft.com/json
	Download:	nuget
